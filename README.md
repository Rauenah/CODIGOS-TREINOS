# 🐍 Repositório de Exercícios em Python

Este repositório reúne uma coleção de exercícios práticos em Python, desenvolvidos com o objetivo de consolidar fundamentos da linguagem e lógica de programação.

## 📚 Sobre o Projeto

O projeto tem por objetivo fazer **100 exercícios**, abordando desde conceitos básicos até estruturas mais elaboradas, como:

* Condições simples e aninhadas
* Estruturas de repetição
* Manipulação de strings
* Operações matemáticas
* Listas e sorteios
* Conversores (moeda, temperatura, medidas, base numérica, fuso horário)
* Pequenos sistemas práticos (agenda, jogos, simuladores)

Os exercícios foram desenvolvidos como prática contínua de aprendizado, com foco em lógica, clareza de código e resolução de problemas.

---

## 🧠 Principais Temas Abordados

### 🔹 Lógica de Programação

* Antecessor e sucessor

* Média aritmética

### 🔹 Matemática e Cálculos

* Tabuada
* Raiz quadrada

* Conversões diversas

### 🔹 Condições e Regras de Negócio

* Aprovação de empréstimo
* Alistamento militar
* Classificação de atletas
* Ano bissexto

### 🔹 Manipulação de Texto

* Analisador de texto
* Busca de strings
* Primeira e última ocorrência
* Separação de dígitos

### 🔹 Projetos Simples

* 📅 Agenda de compromissos
* 🎯 Jogo de adivinhação
* 🚗 Simulação de aluguel de carros
* ✈️ Cálculo de custo de viagem
* 🌍 Conversor de fuso horário

## 📁 Estrutura do Repositório

Os arquivos estão organizados por exercícios individuais, cada um focado em um problema específico.
Alguns exemplos:

* `analisador_de_texto22.py`
* `conversor_de_moeda010.py`
* `jogo_adivinação28.py`
* `agenda compromissos simples.py`
* `aprovando_emprestimo36.py`

Também há arquivos auxiliares como:

* Planilhas (`.xlsx`)
* Documentações (`.md`, `.txt`)
* Arquivos de mídia (`.mp3`)
