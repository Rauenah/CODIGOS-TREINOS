Manipulação de texto em Python significa trabalhar com strings (cadeias de caracteres) para analisar, transformar ou formatar informações escritas. É uma das tarefas mais comuns na linguagem, usada em praticamente qualquer aplicação.


Para que serve

Você usa manipulação de texto em Python para:

• 	Limpar dados vindos de arquivos, bancos ou APIs.
• 	Extrair informações específicas (ex.: pegar apenas o domínio de um e-mail).
• 	Formatar mensagens para exibição.
• 	Processar textos em tarefas de NLP (Processamento de Linguagem Natural).


O que envolve manipulação de texto

•Criação de strings 
•Concatenação (juntar textos) 
•Fatiamento (slicing)   
•Alteração de maiúsculas/minúsculas   
•Remoção de espaços   
•Substituição de partes   
•Divisão em lista   
•Busca de padrões  retorna posição
